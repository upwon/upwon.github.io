Vivado2018安装教程

发表于 2020-01-07 更新于 2020-02-11 分类于 教程 阅读次数：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

本文介绍了如何安装Vivado2018💻

软件下载：

登录其官方网站,找到需要下载的文件；

官网链接如下：

https://china.xilinx.com/support/download/index.html/content/xilinx/zh/downloadNav/vivado-design-tools.html

或者直接百度找网盘里面的资源；（也可以只用来下载License）;

下载下来就选择安装;

常见的百度网盘资源搜索网站如下，自己动手丰衣足食

安装过程

点击“xsetup”进行安装；

开始界面；

不更新，点击“Continue”；

点击“Next”；

勾选“I Agree”,点击“Next”；

勾选“Vivado HL System Edition ”,点击“Next”；

Vivado HL Design Edition与 Vivado HL System Edition 对比：

Vivado HL Design Edition includes the full complement of Vivado Design Suite tools for design, including C-based design with Vivado High-Level Synthesis, implementation, verification and device programming. Complete device support, cable drivers and Documentation Navigator are included. Users can optionally add Model Composer to this installation.

译文：Vivado HL Design Edition包括用于设计的Vivado Design Suite工具的完整补充，包括具有Vivado高级综合，实现，验证和设备编程的基于C的设计。 包括完整的设备支持，电缆驱动程序和Documentation Navigator。 用户可以选择将Model Composer添加到此安装中。

Vivado HL System Edition is a superset of Vivado HL Design Edition with the addition of System Generator for DSP. Complete device support, cable drivers and Documentation Navigator are included. Users can optionally add Model Composer to this installation.

译文：Vivado HL System Edition是Vivado HL Design Edition的超集，其中增加了System Generator for DSP。 包括完整的设备支持，电缆驱动程序和Documentation Navigator。 用户可以选择将Model Composer添加到此安装中。

选择默认设置；分别为安装工具，支持的芯片系列，安装选项；

选择安装路径；

创建安装目录；点击“Yes”

查看安装信息；点击“Install”

开始安装；

安装过程；

加载License，继续安装：

文件安装完成；这一步之后会弹出其他的窗口

安装成功；

中间安装过程中弹出其他窗口：

一、安装WinPcap

弹出安装WinPcap的窗口：

如果之前已经安装过WinPcap，会出现已经安装过的提示；

二、加载Licence

点击“Load Licence”,文件名为“vivado.lic”；

Licence安装成功；

二、为系统选择匹配的Matlab版本安装，点击OK；

此时整个安装过程结束；

可以寻找一些Demo来实践啦！
