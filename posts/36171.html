文档/文件关键词内容搜索

发表于 2020-10-18 更新于 2022-11-15 分类于 教程 ， 善用佳软 阅读次数： 本文字数： 1.5k 阅读时长 ≈ 1 分钟

大家都知道要想快速的搜索文件可以使用 Everything ，然而要想搜索文档文件中的内容 Everything 却做不到，那么搜索文档内容该怎么办呢？AnyTXT Searcher 和 BBdoc 是两款搜索文档内容的软件。🔍


AnyTXT Searcher

下载地址： https://anytxt.net/

AnyTXT Searcher　是一款带有强大全文搜索引擎的桌面搜索工具。

AnyTXT Searcher 有一个内置的强大的文档解析引擎，它可以在不安装任何其他软件的情况下提取常用文档的文本，并结合内置的高速索引系统来存储文本的元数据。使用 AnyTXT 搜索器，可以快速查找计算机上存在的任何文字。它在 Windows 10,8,7，Vista，XP，2008,2012,2016等操作系统上运行良好。

支持的格式

  • Plain Text Format (txt, cpp, html, etc.) 纯文本格式(txt、 cpp、 html 等)
  • Microsoft Outlook (eml) Microsoftoutlook (eml)
  • Microsoft Word (doc, docx) Microsoftword (doc，docx)
  • Microsoft Excel (xls, xlsx) Microsoftexcel (xls，xlsx)
  • Microsoft PowerPoint (ppt, pptx) Microsoft PowerPoint (ppt，ppt)
  • Portable Document Format (pdf) (beta) 便携式文件格式(pdf)(beta)
  • eBook Formats(mobi, epub, etc.) 电子书格式(mobi、 epub 等)

也可以新增其它格式。

首次安装后会需要一定的时间进行本地数据库的构建，过一会我们就可以进行搜索了。

以搜索“多载波”为例，在①中输入要搜索的关键词或者句子，在②中可以只能要搜索哪一个磁盘，在③中可以指定文件类型，然后点击④即可以开始搜索。

缩小检索的磁盘为　Ｄ盘。

再缩小检索的文件类型为　docx 。

若没有想要的文件类型，可以新增一个，比如 新增一个 .cpp

选择添加文件类型。

然后选择一个 CPP文件，在下面的弹窗中选择 所以文件将被包含。

以后就可以搜索CPP文件的关键词啦。


BBdoc 文档搜索

下载地址： http://www.bbdoc.cn/

bbdoc 是一款可以搜索文档内容的小软件。

  • 支持文件名，文档内容快速搜索，
  • 支持文件类型、文件修改时间筛选

下载后，解压并运行 bbdoc.exe 即可使用。首次开启，bbdoc 先会花一段时间来建立索引。在搜索框中输入你想要搜的内容，然后按下回车键，所有内含相关字段的文档，都被被罗列出来。当然也可以指定搜索文档的类型，一般得办公文件的类型都支持。缺点是其不支持新增文件类型。

软件主要功能介绍：

1、支持按照文档类型搜索，目前支持word、pdf、Excel、压缩文件、csv、wps、txt等，针对图片、音视频、html等文件搜索会升级上来，更方便查询管理。

2、支持按照文件大小搜索，也支持按照文件大小排序。

3、支持按照文档修改或创建时间段查询，也支持按照日期排序。

4、支持按照文件名关键词搜索。

5、支持按照文档内容快速搜索，升级会增加2-3个关键词组合搜索。

6、搜索结果支持：1）直接打开文档内容 2）直接进入文档所在目录 3）支持查看文档内容搜索快照预览。

结论

  1. 使用 AnyTXT Searcher 似乎搜索的内容比 BBdoc 文档搜索 更齐全一些。
  2. 使用 AnyTXT Searcher 可以自定义文档类型（文档后缀）。
  3. BBdoc 文档搜索 UI漂亮一些。
