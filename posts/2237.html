1гҖҒ жј”и®ІжҜ”иөӣзЁӢеәҸйңҖжұӮ

1.1 жҜ”иөӣи§„еҲҷ

  • еӯҰж ЎдёҫиЎҢдёҖеңәжј”и®ІжҜ”иөӣпјҢе…ұжңү12дёӘдәәеҸӮеҠ гҖӮжҜ”иөӣе…ұдёӨиҪ®пјҢз¬¬дёҖиҪ®дёәж·ҳжұ°иөӣпјҢз¬¬дәҢиҪ®дёәеҶіиөӣгҖӮ
  • жҜ”иөӣж–№ејҸпјҡеҲҶз»„жҜ”иөӣпјҢжҜҸз»„6дёӘдәәпјӣйҖүжүӢжҜҸж¬ЎиҰҒйҡҸжңәеҲҶз»„пјҢиҝӣиЎҢжҜ”иөӣ
  • жҜҸеҗҚйҖүжүӢйғҪжңүеҜ№еә”зҡ„зј–еҸ·пјҢеҰӮ 10001 ~ 10012
  • з¬¬дёҖиҪ®еҲҶдёәдёӨдёӘе°Ҹз»„пјҢжҜҸз»„6дёӘдәәгҖӮ ж•ҙдҪ“жҢүз…§йҖүжүӢзј–еҸ·иҝӣиЎҢжҠҪзӯҫеҗҺйЎәеәҸжј”и®ІгҖӮ
  • еҪ“е°Ҹз»„жј”и®Іе®ҢеҗҺпјҢж·ҳжұ°з»„еҶ…жҺ’еҗҚжңҖеҗҺзҡ„дёүдёӘйҖүжүӢпјҢеүҚдёүеҗҚжҷӢзә§пјҢиҝӣе…ҘдёӢдёҖиҪ®зҡ„жҜ”иөӣгҖӮ
  • з¬¬дәҢиҪ®дёәеҶіиөӣпјҢеүҚдёүеҗҚиғңеҮә
  • жҜҸиҪ®жҜ”иөӣиҝҮеҗҺйңҖиҰҒжҳҫзӨәжҷӢзә§йҖүжүӢзҡ„дҝЎжҒҜ

1.2 зЁӢеәҸеҠҹиғҪ

  • ејҖе§Ӣжј”и®ІжҜ”иөӣпјҡе®ҢжҲҗж•ҙеұҠжҜ”иөӣзҡ„жөҒзЁӢпјҢжҜҸдёӘжҜ”иөӣйҳ¶ж®өйңҖиҰҒз»ҷз”ЁжҲ·дёҖдёӘжҸҗзӨәпјҢз”ЁжҲ·жҢүд»»ж„Ҹй”®еҗҺз»§з»ӯдёӢдёҖдёӘйҳ¶ж®ө
  • жҹҘзңӢеҫҖеұҠи®°еҪ•пјҡжҹҘзңӢд№ӢеүҚжҜ”иөӣеүҚдёүеҗҚз»“жһңпјҢжҜҸж¬ЎжҜ”иөӣйғҪдјҡи®°еҪ•еҲ°ж–Үд»¶дёӯпјҢж–Үд»¶з”Ё.csvеҗҺзјҖеҗҚдҝқеӯҳ
  • жё…з©әжҜ”иөӣи®°еҪ•пјҡе°Ҷж–Үд»¶дёӯж•°жҚ®жё…з©ә
  • йҖҖеҮәжҜ”иөӣзЁӢеәҸпјҡеҸҜд»ҘйҖҖеҮәеҪ“еүҚзЁӢеәҸ

  • иҮіжӯӨпјҢйЎ№зӣ®е·ІеҲӣе»әе®ҢжҜ•

6гҖҒжј”и®ІжҜ”иөӣеҠҹиғҪ

6.1 еҠҹиғҪеҲҶжһҗ

жҜ”иөӣжөҒзЁӢеҲҶжһҗпјҡ

жҠҪзӯҫ вҶ’ ејҖе§Ӣжј”и®ІжҜ”иөӣ вҶ’ жҳҫзӨәз¬¬дёҖиҪ®жҜ”иөӣз»“жһң вҶ’

жҠҪзӯҫ вҶ’ ејҖе§Ӣжј”и®ІжҜ”иөӣ вҶ’ жҳҫзӨәеүҚдёүеҗҚз»“жһң вҶ’ дҝқеӯҳеҲҶж•°

6.2 еҲӣе»әйҖүжүӢзұ»

  • йҖүжүӢзұ»дёӯзҡ„еұһжҖ§еҢ…еҗ«пјҡйҖүжүӢе§“еҗҚгҖҒеҲҶж•°
  • еӨҙж–Үд»¶дёӯеҲӣе»ә speaker.hж–Үд»¶пјҢе№¶ж·»еҠ д»Јз Ғпјҡ

6.3 жҜ”иөӣ

6.3.1 жҲҗе‘ҳеұһжҖ§ж·»еҠ 

  • еңЁspeechManager.hдёӯж·»еҠ еұһжҖ§

6.3.2 еҲқе§ӢеҢ–еұһжҖ§

  • еңЁspeechManager.hдёӯжҸҗдҫӣејҖе§ӢжҜ”иөӣзҡ„зҡ„жҲҗе‘ҳеҮҪж•° void initSpeech();
  • еңЁspeechManager.cppдёӯе®һзҺ°void initSpeech();
  • SpeechManagerжһ„йҖ еҮҪж•°дёӯи°ғз”Ёvoid initSpeech();

6.3.3 еҲӣе»әйҖүжүӢ

  • еңЁspeechManager.hдёӯжҸҗдҫӣејҖе§ӢжҜ”иөӣзҡ„зҡ„жҲҗе‘ҳеҮҪж•° void createSpeaker();
  • еңЁspeechManager.cppдёӯе®һзҺ°void createSpeaker();
  • SpeechManagerзұ»зҡ„ жһ„йҖ еҮҪж•°дёӯи°ғз”Ёvoid createSpeaker();
  • жөӢиҜ• еңЁmainеҮҪж•°дёӯпјҢеҸҜд»ҘеңЁеҲӣе»әе®Ңз®ЎзҗҶеҜ№иұЎеҗҺпјҢдҪҝз”ЁдёӢеҲ—д»Јз ҒжөӢиҜ•12еҗҚйҖүжүӢеҲқе§ӢзҠ¶жҖҒ

  • жөӢиҜ•ж•ҲжһңеҰӮеӣҫпјҡ

  • жөӢиҜ•е®ҢжҜ•еҗҺпјҢеҸҜд»Ҙе°ҶжөӢиҜ•д»Јз ҒеҲ йҷӨжҲ–жіЁйҮҠгҖӮ

6.3.4 ејҖе§ӢжҜ”иөӣжҲҗе‘ҳеҮҪж•°ж·»еҠ 

  • еңЁspeechManager.hдёӯжҸҗдҫӣејҖе§ӢжҜ”иөӣзҡ„зҡ„жҲҗе‘ҳеҮҪж•° void startSpeech();
  • иҜҘеҮҪж•°еҠҹиғҪжҳҜдё»иҰҒжҺ§еҲ¶жҜ”иөӣзҡ„жөҒзЁӢ
  • еңЁspeechManager.cppдёӯе°ҶstartSpeechзҡ„з©әе®һзҺ°е…ҲеҶҷе…Ҙ
  • жҲ‘д»¬еҸҜд»Ҙе…Ҳе°Ҷж•ҙдёӘжҜ”иөӣзҡ„жөҒзЁӢ еҶҷеҲ°еҮҪж•°дёӯ

6.3.5 жҠҪзӯҫ

еҠҹиғҪжҸҸиҝ°пјҡ

  • жӯЈејҸжҜ”иөӣеүҚпјҢжүҖжңүйҖүжүӢзҡ„жҜ”иөӣйЎәеәҸйңҖиҰҒжү“д№ұпјҢжҲ‘д»¬еҸӘйңҖиҰҒе°Ҷеӯҳж”ҫйҖүжүӢзј–еҸ·зҡ„е®№еҷЁ жү“д№ұж¬ЎеәҸеҚіеҸҜ
  • еңЁspeechManager.hдёӯжҸҗдҫӣжҠҪзӯҫзҡ„зҡ„жҲҗе‘ҳеҮҪж•° void speechDraw();
  • еңЁspeechManager.cppдёӯе®һзҺ°жҲҗе‘ҳеҮҪж•° void speechDraw();
  • еңЁstartSpeechжҜ”иөӣжөҒзЁӢжҺ§еҲ¶зҡ„еҮҪж•°дёӯпјҢи°ғз”ЁжҠҪзӯҫеҮҪж•°

  • еңЁmainеҮҪж•°дёӯпјҢеҲҶж”Ҝ1йҖүйЎ№дёӯпјҢи°ғз”ЁејҖе§ӢжҜ”иөӣзҡ„жҺҘеҸЈ

  • жөӢиҜ•

6.3.6 ејҖе§ӢжҜ”иөӣ

  • еңЁspeechManager.hдёӯжҸҗдҫӣжҜ”иөӣзҡ„зҡ„жҲҗе‘ҳеҮҪж•° void speechContest();
  • еңЁspeechManager.cppдёӯе®һзҺ°жҲҗе‘ҳеҮҪж•° void speechContest();

  • еҶҚж¬ЎиҝҗиЎҢд»Јз ҒпјҢжөӢиҜ•жҜ”иөӣ

6.3.7 жҳҫзӨәжҜ”иөӣеҲҶж•°

6.4 дҝқеӯҳеҲҶж•°

  • еңЁspeechManager.cppдёӯе®һзҺ°жҲҗе‘ҳеҮҪж•° void saveRecord();
  • еңЁstartSpeechжҜ”иөӣжөҒзЁӢжҺ§еҲ¶зҡ„еҮҪж•°дёӯпјҢжңҖеҗҺи°ғз”Ёдҝқеӯҳи®°еҪ•еҲҶж•°еҮҪж•°

  • жөӢиҜ•пјҢж•ҙдёӘжҜ”иөӣе®ҢжҜ•еҗҺи®°еҪ•дҝқеӯҳжғ…еҶө
