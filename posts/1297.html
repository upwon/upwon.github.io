通信专业网页工具和博客推荐

发表于 2022-10-15 更新于 2023-07-16 分类于 技术 阅读次数： 本文字数： 4k 阅读时长 ≈ 4 分钟

研究生阶段在学习通信(LTE/5G)时偶遇到一些工具网站和不错的博客，在此分享给大家。

工具类

在线绘制 LTE资源网格

http://dhagle.in/LTE

在线绘制 LTE资源网格 ，太棒了吧，还可以选各种参数。

可以根据自己的需求更改 FDD或者TDD 、带宽 、CP类型等等，然后点击 Submit 下方的资源网格就会刷新啦。

网页右上角这边还将不用信道资源的分配用不同颜色区分 ，对应着下方的各个 RE （资源网格上的每个元素称为一个RE Resource Element ，RE 是 LTE 中的最小物理资源，存放一个调制符号）。另外还有饼图列出了各个信道资源的比例，当然 PDSCH 占比是最大的，毕竟要传输业务数据。

在线绘制 LTE资源网格2

http://www.lte-bullets.com/

同样是一个绘制资源网格的工具，可以在 lte-bullets 的网站中下载。

This tool is free to download and use. It models the allocation of downlink Resource Elements to the set of Signals and Physical Channels. The user can configure each of the variables which have an impact upon the allocation of Resource Elements, e.g. the channel bandwidth, number of transmit antenna and cell identity. The tool also quantifies throughput for each modulation scheme and a range of assumed coding rates.
Results from the tool can be exported into a csv file for subsequent viewing in Excel. An Excel Macro can be downloaded to read and format the data.

安装好后，根据需求设置参数，点击左下角的按钮生成即可。

还能计算出吞吐量什么的

还可以导出成 CSV 文件

5G NR/LTE ARFCN-Frequency Caculator (3GPP Specs) – TechSpec (itecspec.com)

在线信道编码误码率 比较器

https://aff3ct.github.io/comparator.html?curve0=162fd1d&curve1=de60d27&xaxis=Eb%2FN0&yaxes=BER%2CFER

This page is an online tool dedicated to the Bit Error Rate (BER) and Frame Error Rate (FER) curve comparisons. Various type of codes are considered like the Turbo codes (LTE, LTE-Advanced, CCSDS, etc.), the Low-Density Parity-Check (LDPC) codes (5G, Wi-Fi, WiMAX, CCSDS, WRAN, DVB-S2, etc.), and the more recently introduced Polar codes (5G).

本页面是一个在线工具，专门用于比特错误率(BER)和帧错误率(FER)曲线比较。 各种类型的码被认为像 Turbo 码(LTE，LTE-advanced，CCSDS 等) ，低密度奇偶校验(LDPC)码(5 g，Wi-Fi，WiMAX，CCSDS，wlan，DVB-S2等) ，以及最近引入的极性码(5 g)。

The presented results have been simulated with AFF3CT: our Open-source communication chain dedicated to the Forward Error Correction (FEC) simulations. The simulated output text files are freely available on our dedicated GitHub repository.

给出的结果已经被 AFF3CT 模拟: 我们的开源通信链专门用于前向错误更正通信系统(FEC)模拟。 模拟输出的文本文件可以在我们专用的 GitHub 存储库中免费获得。

Do you like the BER/FER Comparator? Is it useful in your research works? If yes, you can thank us by citing the following journal article: A. Cassagne et al., “AFF3CT: A Fast Forward Error Correction Toolbox!,“ *SoftwareX*, 2019.

你喜欢 ber / fer 比较器吗？ 它在你的研究工作中有用吗？ 如果答案是肯定的，你可以通过引用下面的期刊文章来感谢我们: a. Cassagne et A.. ，“ AFF3CT: 一个快速的前向错误更正工具箱！ ” SoftwareX，2019。
